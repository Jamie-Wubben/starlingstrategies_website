## Challenge
Crear una aplicación web rápida y multilingüe capaz de transcribir con precisión archivos de audio y video, y convertir instantáneamente el resultado en archivos de subtítulos SRT formateados correctamente.

## Solution
Se desarrolló una herramienta web full-stack utilizando un modelo de IA de alto rendimiento para la transcripción, junto con una interfaz intuitiva de arrastrar y soltar. La solución es compatible con varios formatos de archivo y múltiples idiomas.

## Outcome
Una utilidad altamente eficiente y fácil de usar que reduce drásticamente el tiempo y el esfuerzo necesarios para generar subtítulos precisos para contenido multimedia.
