## Challenge
To build a fast, multi-language web application capable of accurately transcribing audio and video files and instantly converting the output into correctly formatted SRT subtitle files.

## Solution
Developed a full-stack web tool utilizing a high-performance AI model for transcription, paired with an intuitive drag-and-drop interface. The solution supports various file formats and multiple languages.

## Outcome
A highly efficient, user-friendly utility that drastically reduces the time and effort required to generate accurate subtitles for media content.
